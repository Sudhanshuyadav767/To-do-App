# Todo App

A simple and efficient notes application built with React and Vite.

## Features

*   **Create Notes**: Easily add new notes.
*   **View Notes**: See a list of all your notes.
*   **Edit Notes**: Update existing notes.
*   **Delete Notes**: Remove notes from your list.
*   **Modern UI**: Styled with Tailwind CSS for a clean and responsive design.

## Tech Stack

This project is built using the following technologies:

*   [React]
*   [Vite]
*   [Tailwind CSS]
*   [ESLint]
